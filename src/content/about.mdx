## Welcome to Yuki OS

Yuki OS is a retro-futuristic, cyberpunk-themed web-based operating system interface that combines nostalgic computing aesthetics with modern web technologies.

### Features

This interactive desktop environment includes:

- **Retro Desktop Interface**: Classic window management with drag, minimize, maximize, and close functionality
- **Cyberpunk Aesthetics**: Neon gradients, animated backgrounds, and holographic effects
- **Interactive Applications**: Terminal, file browser, AI assistant, and more
- **Real-time Widgets**: Cryptocurrency price tracking and system information
- **Sound System**: Toggle-able sound effects for enhanced interactivity
- **Theme Support**: Switch between dark and light modes
- **Easter Eggs**: Hidden features and surprises throughout the system

### Technology Stack

Built with modern web technologies:

- **Astro**: Static site generation framework
- **TypeScript**: Type-safe JavaScript
- **Tailwind CSS**: Utility-first CSS framework with custom animations
- **MDX**: Enhanced markdown for content

### About This Project

This project started as an exploration of creative web design, combining the nostalgic feel of retro operating systems with the sleek aesthetics of cyberpunk culture. The goal was to create an immersive, interactive experience that showcases what's possible with modern web technologies.

Feel free to explore the desktop environment and discover all the hidden features!
